Overview

A 2D autonomous navigation simulator built in C++17. The agent navigates a construction site obstacle map using RRT* (Rapidly-exploring Random Tree Star), following a collision-free path while streaming live telemetry to disk at 10Hz.

How It Works

RRT* planner

The core algorithm implements full RRT* with rewiring. Unlike basic RRT, RRT* continuously improves the tree by rewiring nodes to shorter paths as new samples are added. Goal-biased sampling guides exploration toward the target with 15% probability, balancing exploration and convergence. A greedy shortcut post-processor walks the final path and attempts direct connections between non-adjacent nodes, typically reducing path length by ~30%.

telemetry logger

The simulator outputs telemetry at 10Hz to CSV files, logging position (x, y), heading (radians), speed (pixels/sec), and distance traveled. A live HUD overlays the same data on screen. Each simulation run creates a timestamped log file for offline analysis.

world grid

The environment is a 40×30 occupancy grid representing a construction site. Obstacles include walls (permanent barriers), debris (irregular shapes), and equipment zones (rectangular hazards). The grid uses pixel-based collision detection for fast queries during tree expansion.

Technical Highlights

  • Full RRT* rewiring implementation — not just basic RRT
  • Greedy shortcut post-processor reduces path length by ~30%
  • Telemetry streams position, heading, speed, distance at 10Hz
  • Press R to replan live with a new random seed
  • Built with SFML 2.6 via CMake FetchContent — single cmake command to build
